# 🎾 Pong Game – COMP 3023

## 📌 Overview
This project is a complete implementation of the classic **Pong** game, developed as part of the COMP 3023 quest-based coursework. It focuses on core mechanics such as real-time input handling, collision physics, and dynamic gameplay behavior.

## ✅ Base Gates Implementation
- Paddle movement with keyboard input and boundary constraints  
- Ball physics with accurate collision response using PhysicsMaterial2D  
- Score tracking system with UI display and goal detection  

## 🚀 Expansion Features
- **Curve Ball Mechanic:** Ball trajectory dynamically adjusts based on point of contact with the paddle  
- **AI Opponent:** Automated paddle movement using vector-based tracking of the ball position  

## 🧠 Technical Highlights
- 2D collision detection and response  
- Unity physics system (bounciness and friction control)  
- Real-time input handling  
- Vector-based movement (MoveTowards)  
- UI integration using TextMeshPro
