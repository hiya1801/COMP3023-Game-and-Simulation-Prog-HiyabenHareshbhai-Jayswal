# 🚀 Asteroids – COMP 3023

## 📌 Overview
This project is a complete implementation of the classic **Asteroids** game, developed as part of the COMP 3023 quest-based coursework. It emphasizes physics-based movement, object interaction, and dynamic gameplay systems.

## ✅ Base Gates Implementation
- Ship movement using thrust and rotation (tank controls) with realistic drift  
- Screen wrapping system using viewport transformation  
- Projectile system using prefab instantiation with timed destruction  

## 🚀 Expansion Features
- **Splitting Asteroids:** Large asteroids break into smaller fragments upon destruction  
- **Gravity Well Mechanic:** A central hazard applies force to attract nearby objects  

## 🧠 Technical Highlights
- Rigidbody-based physics and force application  
- Object instantiation and lifecycle management  
- Screen-space to world-space transformations  
- Vector-based motion and rotation  
- Real-time gameplay interactions
